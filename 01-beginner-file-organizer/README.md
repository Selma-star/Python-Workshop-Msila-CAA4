# 📂 Project 1: File Organizer

## 🎯 Goal
Automatically sort files in a folder by extension (`.jpg` → Images, `.pdf` → Documents).

## 🧠 What You'll Learn
- Navigating file systems with Python
- Using `os` and `shutil` libraries
- Loops, conditionals, and lists
